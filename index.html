Introduction

This is a Python 3 module that use the Genetic Algorithm to optimize the features of the indicators used on a trading strategy and backtest it.

Supported Versions:

App Version 5.0.1. Latest update 1/11/2022.

About this App

By feeding any dataframe with historical data for cryptocurrency and by using any of the available optimization algorithms, this app can find the optimal solution to a trading bot. The module comes with a wrapper on the Binance API making the process to fetch the data easier.

Technology

This module is built with Python 3.
The package contains:
- Strategy backtesting - Standalone genetic algorithm
- Non-dominated sorting genetic algorithm II

The purpose of this guide is to illustrate some of the main features that the tradingbot module provides.

Getting Started

Install the App

The github repository for this app is private. Without access the installation is impossible.

  1. Clone, or download the repository as zip and uzip it.

  2. Copy the cloned/uzipped folder to your project folder.

  3. The app is now ready to use.

Important: Make sure to check for updates. There is no auto-update feature just yet.
